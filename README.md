# Магический Шар 8 (Magic 8 Ball)

Проект представляет собой шуточную программу, имитирующую магический шар 8, который предсказывает будущее. Пользователь задает вопрос, а программа случайным образом отвечает на него. Этот проект — веселая имитация классического магического шара, который отвечает на вопросы.

## Описание

Магический шар 8 — это интерактивная программа, которая генерирует случайный ответ на заданный пользователем вопрос. Ответ может быть положительным, отрицательным или нейтральным, и помогает добавить элемент случайности в принятие решений.

### Пример работы программы:

1. Программа предлагает пользователю ввести вопрос.
2. На основе случайного выбора, программа отвечает на вопрос одного из заранее подготовленных ответов.

Программа не предназначена для серьезного использования, а является развлечением.

## Как запустить

Чтобы запустить программу, выполните следующие шаги:

1. Клонируйте репозиторий
2. Перейдите в директорию с проектом
3. Запустите программу
